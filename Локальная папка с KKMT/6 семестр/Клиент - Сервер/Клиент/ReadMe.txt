Наименование: Client_Chat (Чат клиент). Версия программы 1.0.
Применения: Общение в локальной сети.
Программное обеспечение: Windows 7/8/10, не требует стороннего ПО для клиента (не включая Сервер для функционирования). Для сервера необходима установка Java версии выше 1.8.0. Для работы необходимы права на разрешение брандмауэра для открытия порта у ПК.

Программа написана на языке C++ и скомпилирована в C++ Builder 6. Предназначена для общения в локальной сети или средствам передачи пакетов через порты.

Пользователь может передавать текст, цифры, другому подключенному пользователю, так же проверят подключенных людей.
В состав программы входят следующие модули:
– пользовательский интерфейс;
– модуль определения путей;
– модуль принятия и отпраки пакетов;

Интерфейсный модуль построен на двух типах диалогов: диалог "вопрос – ответ" и диалог типа "меню".
Программа эксплуатируется на персональном компьютере (ПК) типа IBM PC/AT. Для работы в диалоговом режиме используется экран дисплея, клавиатура и манипулятор типа "мышь".

Загрузка программы осуществляется путём запуска (exe)файла, который запускает программу.
